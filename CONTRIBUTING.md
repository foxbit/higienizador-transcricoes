# Contribuindo

Obrigado por considerar contribuir! Consulte `PUBLICAR_GITHUB.md` e `INSTALACAO.md`.

## Requisitos
- Python 3.10+
- GTK4 + Libadwaita
- Meson e Ninja
- Flatpak/flatpak-builder (opcional)

## Processo
1. Fork e branch
2. Testes (`pytest`) e lint (`flake8`/`black`/`mypy`)
3. Pull Request com descrição clara
