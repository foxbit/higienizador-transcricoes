# Instalação

Guia completo de instalação para Linux.

## Pré-requisitos
- Python 3.10+
- GTK4 + Libadwaita
- Meson, Ninja
- Flatpak (opcional)

## Passos rápidos
```bash
python build.py setup --profile development
python build.py run --profile development
```

## Configuração da API Gemini
Defina a API Key nas preferências.
